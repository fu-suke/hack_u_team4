# コマンド一覧

## ファイル・ディレクトリ操作系

### `pwd`

現在の作業ディレクトリの絶対パスを表示する。

**標準形式**

```bash
pwd [オプション]
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-L` | 論理パスを表示する。シンボリックリンクをそのまま扱う |
| `-P` | 物理パスを表示する。シンボリックリンクを解決する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `pwd` | 現在のディレクトリを表示する |
| `pwd -P` | シンボリックリンクを解決した物理パスを表示する |

---

### `ls`

ファイルやディレクトリの一覧を表示する。

**標準形式**

```bash
ls [オプション] [ファイルまたはディレクトリ]
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-l` | 詳細情報を表示する |
| `-a` | 隠しファイルも表示する |
| `-h` | ファイルサイズを読みやすい単位で表示する |
| `-R` | サブディレクトリを再帰的に表示する |
| `-S` | ファイルサイズ順に並べる |
| `-t` | 更新時刻順に並べる |
| `-r` | 逆順で表示する |
| `-d` | ディレクトリ自身を表示する |
| `-i` | inode 番号を表示する |
| `--color` | 種類に応じて色付きで表示する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `ls` | カレントディレクトリの一覧を表示する |
| `ls -l /etc` | `/etc` の内容を詳細表示する |
| `ls -la` | 隠しファイルを含めて詳細表示する |
| `ls -lh /var/log` | `/var/log` を読みやすいサイズ表記で詳細表示する |
| `ls -lS` | ファイルサイズが大きい順に詳細表示する |
| `ls -lt` | 更新時刻が新しい順に詳細表示する |
| `ls -R src` | `src` 配下を再帰的に表示する |

---

### `cd`

現在の作業ディレクトリを変更する。

**標準形式**

```bash
cd [ディレクトリ]
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `..` | 1つ上のディレクトリへ移動する |
| `~` | ホームディレクトリへ移動する |
| `-` | 直前にいたディレクトリへ戻る |
| `/path` | 絶対パスで指定したディレクトリへ移動する |
| `relative/path` | 相対パスで指定したディレクトリへ移動する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `cd /etc` | `/etc` に移動する |
| `cd ..` | 1つ上のディレクトリへ移動する |
| `cd ~` | ホームディレクトリへ移動する |
| `cd -` | 直前のディレクトリへ戻る |
| `cd src/app` | `src/app` に移動する |

| `-m` | 作成時の権限を指定する |

**サンプル**

| `mkdir -v data` | 作成結果を表示しながら `data` を作成する |

### `touch`

空ファイルを作成したり、ファイルのタイムスタンプを更新したりする。

**標準形式**

```bash
touch [オプション] ファイル名
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-a` | アクセス時刻だけを更新する |
| `-m` | 更新時刻だけを更新する |
| `-c` | ファイルが存在しない場合に新規作成しない |
| `-t` | 指定した時刻に更新する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `touch memo.txt` | `memo.txt` を作成、またはタイムスタンプを更新する |
| `touch -c memo.txt` | 存在しない場合は作成せず、存在する場合のみ更新する |
| `touch -m app.log` | `app.log` の更新時刻だけを更新する |
| `touch -t 202601011200 file.txt` | `file.txt` の時刻を指定時刻に変更する |

### `cp`

ファイルやディレクトリをコピーする。

**標準形式**

```bash
cp [オプション] コピー元 コピー先
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-r` | ディレクトリを再帰的にコピーする |
| `-R` | ディレクトリを再帰的にコピーする |
| `-i` | 上書き前に確認する |
| `-f` | 強制的に上書きする |
| `-v` | コピー内容を表示する |
| `-p` | 権限やタイムスタンプを保持する |
| `-a` | 可能な限り属性を保持してコピーする |
| `-u` | コピー先が古い場合だけコピーする |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `cp file.txt backup.txt` | `file.txt` を `backup.txt` としてコピーする |
| `cp file.txt backup/` | `file.txt` を `backup` ディレクトリへコピーする |
| `cp -r src backup` | `src` ディレクトリを `backup` に再帰的にコピーする |
| `cp -i file.txt backup.txt` | 上書き前に確認してコピーする |
| `cp -pv app.conf backup.conf` | 属性を保持し、処理内容を表示してコピーする |

### `mv`

ファイルやディレクトリを移動、または名前変更する。

**標準形式**

```bash
mv [オプション] 移動元 移動先
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-i` | 上書き前に確認する |
| `-f` | 強制的に上書きする |
| `-n` | 既存ファイルを上書きしない |
| `-v` | 移動内容を表示する |
| `-u` | 移動先が古い場合だけ移動する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `mv old.txt new.txt` | `old.txt` を `new.txt` に名前変更する |
| `mv file.txt backup/` | `file.txt` を `backup` ディレクトリへ移動する |
| `mv -i old.txt new.txt` | 上書き前に確認して名前変更する |
| `mv -n file.txt backup/` | 既存ファイルを上書きせずに移動する |
| `mv -v app.log logs/` | 処理内容を表示して移動する |

## ファイル内容表示・閲覧系

### `cat`

ファイルの内容を表示または連結する。

**標準形式**

```bash
cat [オプション] ファイル名
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-n` | 全ての行に行番号を付ける |
| `-b` | 空行以外に行番号を付ける |
| `-s` | 連続する空行を1行にまとめる |
| `-A` | 制御文字などを可視化する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `cat file.txt` | `file.txt` の内容を表示する |

| `cat file1.txt file2.txt` | 2つのファイルを連結して表示する |
| `cat -s memo.txt` | 連続する空行をまとめて表示する |

## テキスト検索・加工系

### `grep`

ファイルや標準入力から指定した文字列・パターンを含む行を検索する。

**標準形式**

```bash
grep [オプション] パターン ファイル名
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-i` | 大文字小文字を区別しない |
| `-v` | 一致しない行を表示する |
| `-n` | 行番号を表示する |
| `-r` | ディレクトリを再帰的に検索する |
| `-R` | シンボリックリンクも含めて再帰的に検索する |
| `-l` | 一致したファイル名だけを表示する |
| `-c` | 一致した行数を表示する |
| `-E` | 拡張正規表現を使う |
| `-F` | 固定文字列として検索する |
| `-w` | 単語単位で一致させる |
| `-x` | 行全体が一致するものだけ表示する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `grep "error" app.log` | `app.log` から `error` を含む行を表示する |
| `grep -i "error" app.log` | 大文字小文字を区別せず検索する |
| `grep -n "error" app.log` | 行番号付きで検索結果を表示する |
| `grep -v "debug" app.log` | `debug` を含まない行を表示する |
| `grep -r "TODO" src` | `src` 配下から `TODO` を再帰的に検索する |
| `grep -c "error" app.log` | `error` を含む行数を表示する |

---

### `find`

指定したディレクトリ配下から条件に合うファイルやディレクトリを検索する。

**標準形式**

```bash
find 検索開始ディレクトリ [条件] [処理]
```

**パラメータ一覧**

| パラメータ | 機能 |
|---|---|
| `-name` | ファイル名のパターンで検索する |
| `-iname` | 大文字小文字を区別せずファイル名で検索する |
| `-type f` | 通常ファイルを検索する |
| `-type d` | ディレクトリを検索する |
| `-size` | サイズで検索する |
| `-mtime` | 更新日数で検索する |
| `-atime` | アクセス日数で検索する |
| `-user` | 所有者で検索する |
| `-group` | 所属グループで検索する |
| `-perm` | 権限で検索する |
| `-empty` | 空ファイルまたは空ディレクトリを検索する |
| `-maxdepth` | 検索する最大階層を指定する |
| `-mindepth` | 検索する最小階層を指定する |
| `-exec` | 見つかった対象にコマンドを実行する |
| `-delete` | 見つかった対象を削除する |

**サンプル**

| コマンド例 | 機能 |
|---|---|
| `find . -name "*.log"` | カレントディレクトリ配下から `.log` ファイルを検索する |
| `find /var/log -name "*.log"` | `/var/log` 配下から `.log` ファイルを検索する |
| `find . -type f` | 通常ファイルだけを検索する |
| `find . -type d` | ディレクトリだけを検索する |
| `find . -type f -name "*.txt"` | `.txt` の通常ファイルを検索する |
| `find . -mtime -7` | 7日以内に更新されたファイルを検索する |
| `find . -size +100M` | 100MBより大きいファイルを検索する |
| `find . -empty` | 空のファイルやディレクトリを検索する |

sort [オプション] ファイル名

## 作問時の注意

- 同じ意味になる別コマンドをダミーに入れすぎると、タイプB別解が生まれやすい。
- `tar` は `-c -z -f` のように分割すると別解が増えるため、`-czf` のようにまとめた選択肢にする方が扱いやすい。
- `scp` のポート指定は `-P`、`ssh` のポート指定は `-p` であり、干渉選択肢として有効である。
- `find . -name "*.log"` の `"*.log"` はクォートの有無が重要であり、Lv3 のダミーとして使いやすい。
- 任意順オプションが多い問題は、正解パターンが増えすぎるため避ける。
