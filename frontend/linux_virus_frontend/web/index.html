コマンド表記

コマンドは最大10件まで登録できます。/focus のような文字列や <cmd>+k のようなショートカットを使えます。

文字列コマンドは入力された文字列と完全一致で判定します。

  • <cmd>+v
  • /focus
  • start quiz

修飾キーを使うショートカットは、キーの前に山括弧で指定します。

  • <cmd>+k
  • <ctrl>+j
  • <option>+x
  • <shift>+a
  • <ctrl+option>+<space>
  • <cmd+shift>+p

使用できる修飾キー名は cmd、ctrl、option、shift です。
