Command notation

Text commands match typed text exactly.

  • /focus
  • start quiz

Modifier shortcuts use angle brackets before the key.

  • <cmd>+k
  • <ctrl>+j
  • <option>+x
  • <shift>+a
  • <ctrl+option>+<space>
  • <cmd+shift>+p

Supported modifier names are cmd, ctrl, option, and shift.
